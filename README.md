# React Native Assignment App

React Native Assignment Project with Pattern Lock, Form Inputs, Dropdowns, Date Pickers, and UI Components.

## 📱 About The Project

This project is a React Native based assignment application developed using React Native CLI.

The application includes multiple UI components and functionalities such as pattern lock authentication, text inputs, dropdown selections, date pickers, and form handling screens.

The project was created to practice and demonstrate React Native component handling, user input management, state handling, and mobile UI development concepts.

---

## ✨ Features

- 🔒 Pattern Lock functionality
- 📝 Multiple Text Input fields
- 📅 Date Picker integration
- 📋 Dropdown selection components
- 📱 Mobile-friendly UI screens
- ⚡ Smooth React Native performance
- 🔄 State management using React Hooks
- 🎨 Clean and reusable component structure
